# Customer Stories Images

This directory contains images for customer stories/case studies.

## Image Requirements

### Cover Images
- **Location**: `/public/assets/images/customer-stories/`
- **Naming**: `{company-name}-cover.jpg`
- **Recommended Size**: 1600x1000px (16:10 ratio)
- **Format**: JPG or PNG
- **Examples**:
  - `tibed-dental-cover.jpg`
  - `bfresh-cover.jpg`
  - `farmabox-cover.jpg`

### Implementation/Detail Images
- **Location**: `/public/assets/images/customer-stories/`
- **Naming**: `{company-name}-{number}.jpg`
- **Recommended Size**: 1200x800px or similar
- **Format**: JPG or PNG
- **Examples**:
  - `tibed-dental-1.jpg`
  - `tibed-dental-2.jpg`
  - `tibed-dental-3.jpg`

### Company Logos
- **Location**: `/public/assets/images/clients/`
- **Naming**: `{company-name}-logo.png`
- **Recommended Size**: 400x200px (maintain aspect ratio)
- **Format**: PNG with transparent background preferred
- **Examples**:
  - `tibed-dental-logo.png`
  - `bfresh-logo.png`
  - `farmabox-logo.png`
  - `vido-logo.png`
  - `pelangi-logo.png`
  - `tci-logo.png`
  - `pln-logo.png`

## Tips

- Use high-quality, professional images
- Ensure cover images have good contrast for white text overlay
- Keep file sizes optimized (< 500KB per image)
- Use consistent aspect ratios for a professional look
- Consider adding screenshots of your actual product/system in use
