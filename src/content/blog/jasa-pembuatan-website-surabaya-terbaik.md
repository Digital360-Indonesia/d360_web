## Apa itu Website?

**Website adalah kumpulan halaman web yang saling terhubung dan dapat diakses melalui Internet.** Halaman-halaman web ini biasanya berisi teks, gambar, video, audio, atau elemen lainnya yang dirancang untuk memberikan informasi, hiburan, atau fungsi tertentu kepada pengunjungnya. Setiap website memiliki alamat unik yang disebut URL (Uniform Resource Locator), yang memungkinkan pengguna untuk mengaksesnya menggunakan web browser seperti Google Chrome, Mozilla Firefox, atau Safari.

![Image](/blog/2023/08/Jasa-Website-Surabaya.jpg)Image Source:[ Pinterest.com](https://id.pinterest.com/pin/1108870739484423843/)

Website sendiri terdiri dari dua elemen utama yaitu client side dan server side. **Client side adalah bagian situs yang terlihat oleh audiens melalui browser**, seperti Google Chrome. Bagian ini tersusun atas HTML, CSS, dan JavaScript untuk merancang dan menampilkan halaman web. Sedangkan **server side adalah bagian website yang tersembunyi oleh pengguna, dan berisi file dan data yang diolah web server**. Bagian server-side ini terdiri dari bahasa pemrograman, seperti PHP, Python, atau Ruby on Rails.
